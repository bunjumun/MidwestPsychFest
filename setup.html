— Use satellite view for best detail

Schedule Data

Upload your schedule as a .json file. Export it from Google Sheets using the instructions below, or drop in a file you already have.

How to export from Google Sheets →
Option A — Easiest (use a converter):
1. In your Sheet, go to File → Download → CSV
2. Go to csvjson.com/csv2json and paste the CSV
3. Download the resulting JSON and upload it here

Option B — Direct JSON export:
Use the Export Sheet to JSON Google Sheets add-on

Required column names (case-insensitive):
artist  ·  stage  ·  day (1 or 2)  ·  start (e.g. 14:00)  ·  end  ·  bio (optional)
