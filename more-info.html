Manage FAQ / Info Images

How to add images to this page:
  1. Upload your image file to the assets/img/info/ folder in the GitHub repo.
  2. Enter the filename below (e.g. faq-parking.png) and an optional caption.
  3. Click Add Image — it will appear in the gallery immediately.
  4. The image list is saved in your browser and also in data/info-images.json when you sync.

Tip: For infographics, use tall PNG or JPG images. They display with click-to-enlarge lightbox.
